API Gateway Implementation using node js
=============================================================================

[![License: AGPL v3](https://img.shields.io/badge/License-AGPL%20v3-blue.svg)](http://www.gnu.org/licenses/agpl-3.0)

Logistimo mobile api gateway routes requests to appropriate micro services. Api gateway often handles  request by invoking multiple microservices and aggregating the results. Please visit [www.logistimo.com]  (www.logistimo.com) for further details.

Latest Release
------------------

The most recent release is mobile-api-gateway [1.0.0], released on 2017 September 9th.
